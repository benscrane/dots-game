# Dots Game

A browser-based dots and boxes game built with HTML, CSS, and JavaScript.

## Play the Game

Simply open `index.html` in a web browser to play locally.

## Deployment

This project uses **GitHub Pages** for hosting.

### Live Site

Once deployed, the game will be available at:
```
https://<username>.github.io/dots-game/
```

### How Deployment Works

The site automatically deploys to GitHub Pages when changes are pushed to the `main` or `master` branch.

**Deployment workflow:** `.github/workflows/deploy.yml`

### Setup Instructions (One-time)

To enable GitHub Pages deployment for your fork or copy of this repository:

1. Go to your repository on GitHub
2. Navigate to **Settings** > **Pages**
3. Under "Build and deployment", set **Source** to **GitHub Actions**
4. Push to the `main` or `master` branch to trigger deployment

### Manual Deployment

You can also trigger a deployment manually:

1. Go to the **Actions** tab in your repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click **Run workflow**
